使用rinetd部署矿池地址转发完整指南

2025-04-06

步骤1：SSH登录服务器
ssh root@111.111.111.111 # 输入服务器密码登录

步骤2：安装rinetd
apt update && apt install -y rinetd # 更新源并安装
注意：如果提示权限不足，在命令前加sudo

步骤3：配置转发规则
nano /etc/rinetd.conf # 按Ctrl+O保存，Ctrl+X退出
输入以下内容：

# 格式：监听IP 监听端口 目标地址 目标端口
0.0.0.0 1818 bitcoin.viabtc.io 3333
0.0.0.0 1618 bitcoin.viabtc.io 25  
0.0.0.0 1888 bitcoin.viabtc.io 443
0.0.0.0 1668 ca.zano.luckypool.io 8866
0.0.0.0 1868 usa.hashvault.pro 443
allow *.*.*.*  # 允许所有IP访问
logfile /var/log/rinetd.log  # 启用日志记录

步骤4：开放防火墙端口

ufw allow 1818/tcp  # 开放TCP端口
systemctl restart ufw  # 重启防火墙生效

步骤5：启动转发服务

systemctl restart rinetd  # 重启服务使配置生效
systemctl enable rinetd  # 设置开机自启动

步骤6：验证服务状态
ss -tulnp | grep 1818 # 应显示LISTEN状态
成功示例：
tcp LISTEN 0 128 0.0.0.0:1818 0.0.0.0:* users:(("rinetd",pid=1123,fd=3))

步骤7：测试端口连通性
telnet 111.111.111.111 1818 # 本地电脑执行测试
预期结果：显示Connected to 111.111.111.111

步骤8：配置矿机参数
将矿机配置中的矿池地址改为：

ViaBTC的BCH和BTC智能地址：stratum+tcp://111.111.111.111:1818
ViaBTC的BCH和BTC智能地址：stratum+tcp://111.111.111.111:1618
ViaBTC的BCH和BTC智能地址：stratum+tcp://111.111.111.111:1888
LuckyPool的Zano地址：stratum+tcp://111.111.111.111:1668
Hashvault的XMR地址：stratum+tcp://111.111.111.111:1868

注意保持钱包地址和矿工名不变

步骤9：监控转发状态
tail -f /var/log/rinetd.log # 实时查看转发日志
正常日志示例：
08/Sep/2025:15:23:01 111.111.111.111:1818 bitcoin.viabtc.io:3333 200 1.5

通过以上步骤，即使没有Linux基础的用户也能在10分钟内完成矿池转发配置。建议每季度检查一次/var/log/rinetd.log文件大小，超过100MB时执行truncate -s 0 /var/log/rinetd.log清理日志。
