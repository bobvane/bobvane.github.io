Oracle Cloud2015新版界面添加IPv6

2025-03-17

IPv6具有比IPv4大得多的编码地址空间，彻底解决了IPv4地址不足的问题。同时，IPv6简化了地址配置、网络重编号和路由器通知等方面，从而提升了网络连接速度。此外，使用IPv6还可提高网络安全性，并降低网络延时。甲骨文Oracle Cloud 早就支持使用IPv6的资源。这篇文章中，我将引导您完成在最新版界面 Oracle 云环境中启用 IPv6 所需的所有步骤以及如何为您的 VPS 分配 IPV6 地址。

一、给VCN添加 IPv6 CIDR块

导航菜单–》网络–》虚拟云网络–》点击查看网络详情

点击左侧 CIDR 块/前缀–》添加 CIDR 块/IPv6 前缀

勾选 分配 Oracle 分配的 IPv6 /56 前缀 –》添加 CIDR 块/IPv6 前缀

二、子网启用IPV6 CIDR块


勾选 启用IPV6 CIDR块，输入任意 00-FF 之间的十六进制字符，例如：aa 点击保存！

三、添加路由规则

您必须修改路由表才能通过 IPv6 访问任何内容。

打开 路由表 -》 路由表详情 -》添加路由规则


勾选：IPv6
目标类型：Internet网关
目的地 CIDR 块：::/0

四、配置安全组

入站规则

出站规则

五、给实例配置ipv6

查看服务器实例详情 , 选择你要添加IPv6的实例然后打开 – 附加的 VNIC – VNIC 详细信息 – IPv6 地址



至此网站上的配置就配置完了，只需在机器上重新获取IP即可使用IPv6了。
