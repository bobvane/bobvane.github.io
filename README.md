# 🔁 博客已迁移至新站点

本仓库仅用于将 **[bobvane.github.io](https://bobvane.github.io)** 自动跳转至新的个人博客域名：

👉 **[https://bobvane.top/](https://bobvane.top/)**

---

### 📖 说明
原 GitHub Pages 博客内容已迁移至独立站点，使用 Notion + Vercel + [NotionNext](https://github.com/tangly1024/NotionNext) 搭建。  
本仓库保留是为了保持旧链接可用，并引导访客前往新博客。

---

### 🛠 技术实现
- 仓库仅包含一个 `index.html` 文件；
- 页面自动跳转（`meta refresh` + `window.location.replace()` 双保险）；
- 添加了 SEO 限制标签，避免旧域名继续被搜索引擎索引。
